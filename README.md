# TraderForPoe
This tool makes trading in Path of Exile easy.

## Getting Started

### Prerequisites

What things you need to install the software and how to install them
.

```
.NET Framework 4.6.1
```

### Installing

```
Just download the .zip file, extract it and start TraderForPoe.exe file.
```
